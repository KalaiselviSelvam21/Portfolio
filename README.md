# Kalaiselvi S — Personal Portfolio

A modern, responsive personal portfolio website built with pure HTML, CSS, and JavaScript — no frameworks, no dependencies.

## 🌐 Live Demo
[View Portfolio](https://kalaiselvi-fullstack-developer.vercel.app/)

## ✨ Features
- Fully responsive design — works on mobile, tablet, and desktop
- Animated hero section with scrolling grid background
- Smooth scroll-reveal animations
- Interactive hamburger menu for mobile
- Sections: About, Experience, Projects, Skills, Achievements, Education, Contact
- Custom Full Stack Developer SVG illustration
- Embedded profile photo — no external image hosting needed

## 🛠️ Built With
- HTML5
- CSS3 (CSS Variables, Grid, Flexbox, Animations)
- Vanilla JavaScript
- Google Fonts — Syne, Fraunces, DM Mono
